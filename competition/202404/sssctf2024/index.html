CVE-2023-49785

ChatGPT-Next-Web 的 API Endpoint /api/cors 上存在 SSRF 漏洞，可被用于从服务器端发起任意网络请求并取得响应内容。

/api/cors用于开放代理，允许未经身份验证的用户通过它发送任意HTTP请求，不过这会存在一个问题：它允许客户端通过服务器端端点访问跨域资源，从而绕过内置的浏览器保护以访问跨域资源。
