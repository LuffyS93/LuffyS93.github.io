# 啥是 OPCACHE 呢😕

它是一个 PHP 扩展，通过保存预编译的脚本字节码到共享内存中来提高 PHP 的执行效率，减少了每次加载和编译 PHP 脚本的开销

理解了如何利用 OPCACHE 之后就开始直接上手
